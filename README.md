## 🚀 Começando

### Pré-requisitos

- [Node.js](https://nodejs.org/)
- [Docker](https://docker.com/) (opcional)
- [Yarn](https://yarnpkg.com/) ou npm

### Instalação

```bash
git clone https://github.com/SEU_USUARIO/SEU_PROJETO.git
cd SEU_PROJETO
yarn install
```

### Rodando o projeto

```bash
yarn start
```

Ou com Docker:

```bash
docker-compose up
```
